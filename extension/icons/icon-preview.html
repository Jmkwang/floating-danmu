Floating Danmu Icons

Right-click each icon and "Save image as" to save as PNG

16x16 (icon16.png)
48x48 (icon48.png)
128x128 (icon128.png)

如何生成PNG图标：

  1. 右键点击上面的图标，选择"图片另存为"
  2. 保存时确保文件名是 icon16.png, icon48.png, icon128.png
  3. 或者使用在线转换工具：convertio.co/svg-png
  4. 也可以使用 ImageMagick: magick convert icon128.svg icon128.png
